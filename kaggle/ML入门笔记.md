### underfitting and overfitting（欠拟合、过度拟合）
- 欠拟合
  - 举例：分类算法的分类不够，数据归类到有限的几个类别，导致后续的预测不准（即便对于训练数据也是如此）
- 过度拟合
  - 举例：分类算法的层级足够多，以至于每个分支下仅有有限的、为数不多的样本，会导致预测失准
- “折中”做法
  - 在欠拟合与过度拟合之间，找到均衡点
    - 找出最小MAE值对应的参数
      - 用不同的参数设置模型、训练模型，得到MAE值
    - 用参数重新初始化、训练模型（用全量数据）

- 随机森林：用了很多树，从每棵树的结果中进行综合，得出最终的预测
