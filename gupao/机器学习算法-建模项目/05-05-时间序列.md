# 0.1. 理解时间序列
- 定义：按时间顺序排列的数据
  - 每个数据与一个时间戳对应
- 4个核心成分
  - ①趋势
    - 定义：数据长期的整体变化方向
  - ②季节性
    - 定义：在固定时间周期内出现的规律性波动
  - ③周期性
    - 定义：非固定周期的波动
  - ④不规则波动
    - 定义：无法用趋势、季节性或周期性解释的随机噪声
    - 处理：通过平滑或模型残差分析来分离
- 常见应用场景
  - 预测未来值
  - 分析历史规律
  - 异常检测
  - 因果关系推断
